## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- OpenRouter API key (for AI insights)

### Installation

```bash
# Clone the repository
git clone https://github.com/mekapilgupta/gdgnoida_albuswand.git
cd aqi-delhi

# Install dependencies
npm install

# Set up environment variables
# Create .env file with:
# VITE_OPENROUTER_API_KEY=your_api_key_here

# Start development server
npm run dev

# Or start with browser auto-open
npm run dev -- --open
```

### Building for Production

```bash
npm run build
npm run preview
```
